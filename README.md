## Featured Projects

### 🛡️ IADS-CN — Intelligent Anomaly Detection System for Campus Networks
> **Capstone Project · INF499**

An AI-powered network security system that classifies traffic as normal or anomalous in real time using a feedforward Artificial Neural Network (ANN), augmented by a rule-based heuristic layer for multi-class anomaly typing.

- Built with **Python, TensorFlow, Flask**, and a **React/Tailwind** dashboard frontend
- Trained on the **NSL-KDD dataset**; achieved strong classification accuracy across attack categories
- Designed as a practical, deployable solution for campus network infrastructure
- Modular architecture with 7 core components: data pipeline, model training, inference engine, heuristics layer, REST API, dashboard, and reporting module

**Tech Stack:** `Python` `TensorFlow` `Flask` `React` `Tailwind CSS` `scikit-learn` `MySQL`

[![View Repo](https://img.shields.io/badge/View_Repository-181717?style=flat-square&logo=github&logoColor=white)](https://github.com/emmanuelaggor/IADS-CN)
